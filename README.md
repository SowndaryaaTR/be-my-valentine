# Be My Valentine

A cute React frontend to ask: **"Will you be my Valentine?"**

It includes:
- A romantic themed landing card
- Floating hearts in the background
- A celebration burst (hearts + confetti) when "Yes" is clicked
- Responsive layout for mobile and desktop

## Tech Stack

- React (Create React App)
- CSS animations

## Run Locally

```bash
npm install
npm start
```

App runs at `http://localhost:3000`.

## Run Tests

```bash
npm test -- --watchAll=false
```

## Build for Production

```bash
npm run build
```

Build output will be in `/build`.

## Customize Text

Main content is in:
- `src/App.js`

Styles and animations are in:
- `src/App.css`
- `src/index.css`

## Deploy (No Backend Needed)

This project is fully frontend/static. You can deploy it directly to:
- Vercel
- Netlify
- GitHub Pages

After deploy, share the live link.
